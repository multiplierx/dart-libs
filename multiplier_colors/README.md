# Multiplier Colors

![Coverage](coverage_badge.svg?sanitize=true)

## Getting Started

Add in pubspec.yml

```YML
  multiplier_colors:
    path: ../dart-libs/multiplier_colors
```

Import

```DART
import 'package:multiplier_colors/multiplier_colors.dart';
```

Use

```DART
...
Container(
  color: MultiplierColors.rubyRed,
),
...
```
